# BlackStorm Logo Assets — Static Pages

Save your logo files here. The 12 static HTML pages reference these paths.

| File | Purpose | Dimensions |
|------|---------|------------|
| `blackstorm-logo.png` | Nav + footer logo | Original / any |
| `og-image.png` | OpenGraph / social preview | 1200 × 630 px |
| `apple-touch-icon.png` | iOS home screen icon | 180 × 180 px |
| `favicon-32.png` | Browser tab fallback | 32 × 32 px |

## Creating og-image.png
Export `og-image.svg` (in this folder) to PNG at 1200×630.
Inkscape: `inkscape og-image.svg --export-png=og-image.png -w 1200 -h 630`
