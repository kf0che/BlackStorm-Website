# BlackStorm Website

A new static marketing website for BlackStorm Security. It includes a responsive landing page, features section, and contact form.

## Files

- `index.html` — main website page
- `styles.css` — site styling and dark theme
- `script.js` — simple contact form behavior

## Usage

1. Open `index.html` in a browser.
2. Or serve the directory with a static web server for local development.
