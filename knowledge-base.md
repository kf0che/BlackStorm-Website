# Knowledge Base

The BlackStorm Knowledge Base is intended to help customers understand network monitoring, alerts, reports, and basic technology concepts in plain language.

Articles should be useful to non-technical business owners while still being accurate enough for MSPs and technical readers.

## Knowledge Base Goals

The Knowledge Base should:

- Explain concepts before giving instructions
- Avoid jargon where possible
- Use short sections and clear headings
- Include screenshots or examples when available
- Mark plan-specific features clearly
- Avoid unsupported compliance or certification language
- Link to support when a customer needs help
